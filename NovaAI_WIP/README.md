# NovaAI_WIP

Отдельная рабочая зона для прототипирования AI-модуля поиска по изображениям в проекте Liquid Photos.

## Назначение

Здесь можно:

- тестировать библиотеки компьютерного зрения и мультимодального поиска;
- сравнивать качество эмбеддингов и ранжирования;
- проверять пайплайны индексации до интеграции в основной проект;
- проводить локальные эксперименты без риска сломать основное приложение.

## Текущее состояние

Сейчас это минимальное standalone Python-приложение с CLI-точкой входа.

Быстрый запуск:

```bash
pip install -e .
python -m nova_ai_wip
```

или после установки editable-режимом:

```bash
pip install -e .
nova-ai-wip
```
