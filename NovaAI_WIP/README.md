# NovaAI_WIP

Отдельная рабочая зона для прототипирования AI-модуля поиска по изображениям в проекте Liquid Photos.

## Назначение

Здесь можно:

- тестировать библиотеки компьютерного зрения и мультимодального поиска;
- сравнивать качество эмбеддингов и ранжирования;
- проверять пайплайны индексации до интеграции в основной проект;
- проводить локальные эксперименты без риска сломать основное приложение.

## Текущее состояние

Сейчас это минимальное standalone Python-приложение с CLI-точкой входа.
В него уже добавлен первый smoke-test для `open_clip_torch`.

Быстрый запуск:

```bash
pip install -e .
python -m nova_ai_wip
```

или после установки editable-режимом:

```bash
pip install -e .
nova-ai-wip
```

## Поиск по текущим векторам

После экспорта `.txt`-эмбеддингов можно искать по изображениям так:

```bash
python -m nova_ai_wip.semantic_search_cli "sea"
```

или:

```bash
nova-ai-search "девушка"
```

Приложение строит embedding для текста, сравнивает его с локальными `.txt`-векторами изображений и выводит top-3 результата.

## GPU-режим

Для NVIDIA GPU здесь проверен запуск OpenCLIP через CUDA-сборки PyTorch.

Команда установки:

```bash
python -m pip install --force-reinstall torch==2.10.0+cu128 torchvision==0.25.0+cu128 --index-url https://download.pytorch.org/whl/cu128
```

После этого `python -m nova_ai_wip` автоматически выбирает `cuda`, если видеокарта доступна.
